# TC-TM-01 — Enable and show TrinketMenu

**Area:** TrinketMenu | **Client:** Era | **Mandatory:** yes

## Preconditions

- Several trinkets in bags and/or equipped

## Steps

1. Open `/rggm opt` → TrinketMenu configuration
2. Enable the TrinketMenu
3. Observe the screen
4. Disable it again, then re-enable for the following TM cases

## Expected

- Enabled: the TrinketMenu frame appears showing all available trinkets and their cooldown
  status
- Disabled: the frame disappears completely
- The enabled state survives `/reload`
